# El Universo - Explora el Cosmos

Página web educativa sobre astronomía con información sobre galaxias, estrellas, planetas y el Big Bang.

## Contenido

- Introducción al universo
- Galaxias: tipos y la Vía Láctea
- Estrellas: características y el Sol
- Planetas del Sistema Solar
- Teoría del Big Bang
- Datos curiosos sobre el espacio

## Instalación

```bash
git clone https://github.com/tu-usuario/el-universo.git
cd el-universo
```

Abre `index.html` en tu navegador.

## Tecnologías

- HTML5
